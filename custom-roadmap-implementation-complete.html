🚀 How to Access the Feature

  1. Navigate to Learning: Click "Learning" in the main navigation
  2. Explore Roadmaps: View curated roadmaps (Frontend, Backend, Full Stack)
  3. Create Custom Roadmap: Click "Create Custom Roadmap" button
  4. Fill the Form:
    • Enter a detailed topic description
    • Select target level (zero-to-advanced, medium-level, advanced-level)
    • Set weekly time investment (1-20 hours)
    • Toggle learning materials inclusion
    • Describe your learning goals
  5. Generate Roadmap: Submit form to create AI-powered learning path
  6. View & Track: Access your roadmap and track progress
